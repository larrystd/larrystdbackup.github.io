C语言中，数据和处理数据的操作（函数)是分开声明的，不支持数据函数之间的关联性，称之为程序性的(procedural)。

1. 对象类型

C++中可以通过独立抽象数据类型实现。

class Point3d 
{ 
public:  
    Point3d( float x = 0.0, float y = 0.0, float z = 0.0 ): 
_x(x), _y(y), _z(z){ } 
 
    float x() { return _x;} 
    float y() { return _y;} 
    float z() { return _z;} 
 
private: 
    float _x; 
    float _y; 
    float _z; 
}; 
 
// 类外部inline函数 
inline ostream& 
operator << (opstream &os, const Point3d &pt ) 
{ 
    os << "(" << pt.x() << ", " << pt.y() << "," << pt.z() << ")" ; 
};
