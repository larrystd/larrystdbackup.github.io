numpy

使用

不同于python列表，numpy要求数组必须包含同一类型的数据。

np.array([1, 2, 3, 4], dtype='float32')

np.zeros(10, dtype=int)	# 长为10的数组，值为0

np.ones((3,5), dtype=float) # 3*5的浮点型数组

np.linspace(0, 1, 5) # 五个元素数组，五个元素在0~1均匀分配

np.arrange(0, 20, 2) # 0开始20结束，步长为2

np.random.seed(0)  # 随机数种子

np.random.random((3, 3))  # 3*3的随机数组,0~1均匀分布

np.random.normal(0, 1, (3,3)) # 3*3, 均值0, 方差1的正态分布随机数组

np.random.randint(0, 10, (3, 3)) # 3*3 [0, 10)区间的随机数组、

np.eye(3) # 3*3单位矩阵

numpy常用类型 bool_ 布尔值 int18 即字节-128~127, int32 四字节整型 uint32 四字节无符号整型， float32 单精度浮点型 float64 双精度浮点型

索引切片, x[start:stop:step]一般格式。
x[:2, :3]

变形reshape
grid = np.arrange(1, 10).reshape((3, 3))

拼接分裂
np.concatenate([x ,y])
二维数组的拼接

grid = np.array([1,2,3], [4,5,6])
np.concatenate([grid, grid])
相当于 np.vstack([grid, grid])
[
	[1,2,3], [4,5,6],[1,2,3], [4,5,6]
]

np.concatenate([grid, grid], axis = 1)

相当于 np.hstack([grid, grid])

[
	[1,2,3, 1, 2, 3],
	[4,5,6,4,5,6]
]

分裂

x = [1, 2, 3, 99, 99, 3, 2, 1]
x1,x2, x3 = np.split(x, [3, 5]) # 记录分裂点的位置

通用函数

numpy数组的操作会默认执行到数组的每个元素。

x = np.arrage(4)
[0, 1, 2, 3]
x+5
[5, 6, 7, 8]
x/2
[0, 0.5, 1., 1.5]
x // 2
[0, 0, 1, 1]
x**2
[0, 1, 4, 9]

np.abs() 绝对值
np.sin()

theta = np.linspace(0, np.pi, 3)
np.sin(theta)

np.exp(x)	# e^x
np.log(x)	# ln(x)
np.log2(x)	# log2(x)

np.sum
np.min
np.mean
np.std	# 标准差
np.var # 方差
np.argmax # 最大元素的索引
np.sort() # 默认快速排序

numpy的一个原则是能向量运算就向量运算，可以广播，这样可以减少循环带来的效率低下

a = np.ones((2, 3))
[[1, 1, 1], [1, 1, 1]]

b = np.arrange(3)
[0 , 1, 2]

a+b
[[1, 2, 3], [1, 2, 3]]

布尔数据操作

np.sum(x<6)这个语句可以计算x<6的个数，因为true被解释成1, false被解释成0。

布尔数组可以做为掩码x[x < 5]可以把x<5的值从数组中选出。

pandas

pandas可以看成增强版Numpy结构化数组，行列不只是简单的整数索引，还可以带上标签。

Series对象

Series对象是一个带索引数据构成的一维数组,可以用一个数组创建Series对象

data = pd.Series([0.25, 0.5, 0.75, 1.0])

data.values	# 返回array
data.index # 返回Index对象

data[0]
data[1:3] # 可以切片

data.keys()
data.items()

data['a':'c']
data[(data>0.3) & (data < 0.8>)]

Series可以看成特殊的字典

data = pd.Series([0.25, 0.5, 0.75, 1.0], index = ['a', 'b', 'c', 'd'])

data['a']

data = pd.Series({'a':1, 'b':2})

可以直接通过dict来创建Series

DataFrame对象

DataFrame可用认为是二维的Series, 

population = pd.Series(population_dict)
area = pd.Series(area_dict)
states = pd.DataFrame({'population' : population, 'area' : area})

states.index
state.columns # 列标签

states['area'] # 返回第一列

创建方法, 如果有键不存在，会用NAN表示

# 字典列表创建
data = [{'a':i, 'b':2*i} for i in range(2)]
pd.DataFrame(data)

# Series对象字典创建
# Numpy二维数组创建

pd.DataFrame(np.random.rand(3,2), columns=['foo', 'bar'], index=['a', 'b', 'c'])

data['density'] = data['pop'] / data['area']
data.values	# 返回一个二维数组
data.values[0] # 一行数据
data.iloc[:3, :2] # index索引器
data.loc[: 'a'] # tag索引器
data.ix]:3, 'a'] # 混合效果索引器

Index对象可用看作是不可变数组，pd.index, pd.columns返回的都是Index对象，相比于Numpy数组，Index是不可变的，其他基本一致。

缺失值

  • None是一个python单体对象，常在代码中表示缺失值，但由于是一个python对象，不能表示Numpy/Pandas数组类型缺失值，只能用于object数组
  • NaN, 任何数与NaN操作，结果均为NaN。Pandas会将没有标签值的数据自动转为nan

缺失值处理

isnull() #发现缺失值
data.dropna() #剔除缺失值， 默认剔除整行
data.dropna(axis='columns') #剔除整列

fillna() #填充缺失值
filna(method='ffill', axis=1)

pandas除了数值方法，还有字符串方法。一般储存的类型不是数值就是字符串。
